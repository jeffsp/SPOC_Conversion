# NAME

spoc2las - Convert a SPOC format file to a LAS format file

# USAGE

spoc2las [*options*] *filename.spoc* *filename.las*

# DESCRIPTION

Convert a SPOC format file to a LAS format file

# OPTIONS

\-\-help, -h
:   Get help

\-\-verbose, -v
:   Set verbose mode ON

\-\-version, -e
:   Print version information and exit
